# Basic String prototype extension which adds:

1. `.capitalize()`
2. `.allCaps()`
3. `.capitalizeWords()`
4. `.removeExtraSpaces()`
5. `.kabobCase()`
6. `.snakeCase()`
7. `.camelCase()`
